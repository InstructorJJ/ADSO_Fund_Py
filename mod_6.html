🎯 1. Introducción a las Estructuras de Datos

▼

Las estructuras de datos son formas organizadas de almacenar y manipular información en nuestros programas. En Python, tenemos tres estructuras fundamentales:

Estructura Características Cuándo usar
Listas Ordenadas, modificables, permiten duplicados Colecciones que pueden cambiar
Tuplas Ordenadas, inmutables, permiten duplicados Datos que no deben cambiar
Diccionarios No ordenados*, modificables, claves únicas Relaciones clave-valor

🌟 Analogía del Mundo Real

Lista: Como una lista de compras que puedes modificar

Tupla: Como las coordenadas de tu casa (no cambian)

Diccionario: Como una agenda telefónica (nombre → teléfono)

📋 2. Listas: Colecciones Dinámicas

▼

Las listas son la estructura de datos más versátil en Python. Pueden contener diferentes tipos de datos y se pueden modificar.

Creación y Acceso

# Crear listas estudiantes = ["Ana", "Carlos", "Diana", "Eduardo"] calificaciones = [85, 92, 78, 94] mixta = ["Python", 3.9, True, [1, 2, 3]] # Acceso por índice print(estudiantes[0]) # Ana print(estudiantes[-1]) # Eduardo (último elemento) # Slicing (rebanadas) print(estudiantes[1:3]) # ["Carlos", "Diana"]

# Lista de tareas para un desarrollador tareas = ["Diseñar interfaz", "Escribir código", "Hacer pruebas"] # Agregar elementos tareas.append("Documentar") # Al final tareas.insert(1, "Revisar requisitos") # En posición específica # Eliminar elementos tareas.remove("Hacer pruebas") # Por valor tarea_eliminada = tareas.pop() # Último elemento tarea_eliminada = tareas.pop(0) # Por índice # Buscar y contar posicion = tareas.index("Escribir código") # Encuentra posición cantidad = tareas.count("Documentar") # Cuenta ocurrencias # Ordenar calificaciones = [85, 92, 78, 94, 88] calificaciones.sort() # Orden ascendente calificaciones.sort(reverse=True) # Orden descendente

🔒 3. Tuplas: Datos Inmutables

▼

Las tuplas son colecciones ordenadas e inmutables. Perfectas para datos que no deben cambiar.

# Crear tuplas coordenadas = (10, 20) persona = ("Ana", 25, "Desarrolladora") colores_rgb = (255, 128, 64) # Una tupla de un solo elemento necesita coma singleton = (42,) # Acceso igual que las listas print(coordenadas[0]) # 10 print(persona[1:]) # (25, "Desarrolladora") # Desempaquetado de tuplas nombre, edad, profesion = persona x, y = coordenadas

🗝️ 4. Diccionarios: Mapeo Clave-Valor

▼

Los diccionarios almacenan datos en pares clave-valor, permitiendo acceso rápido y eficiente.

Creación y Uso Básico

# Crear diccionarios estudiante = { "nombre": "Ana García", "edad": 22, "carrera": "Ingeniería de Software", "calificaciones": [85, 92, 88, 95] } # Diferentes formas de crear productos = dict(laptop=1200, mouse=25, teclado=75) vacio = {} # Acceso a valores print(estudiante["nombre"]) # Ana García print(estudiante.get("edad")) # 22 print(estudiante.get("telefono", "No disponible")) # Valor por defecto # Modificar valores estudiante["edad"] = 23 estudiante["telefono"] = "555-0123" # Agregar nueva clave

Métodos Útiles de Diccionarios

# Base de datos de productos inventario = { "laptop": {"precio": 1200, "stock": 15, "categoria": "electrónicos"}, "mouse": {"precio": 25, "stock": 50, "categoria": "accesorios"}, "teclado": {"precio": 75, "stock": 30, "categoria": "accesorios"} } # Métodos de acceso claves = inventario.keys() # Todas las claves valores = inventario.values() # Todos los valores items = inventario.items() # Pares clave-valor # Iterar sobre diccionarios for producto, info in inventario.items(): print(f"{producto}: ${info['precio']}") # Métodos de modificación inventario.update({"tablet": {"precio": 300, "stock": 20}}) info_mouse = inventario.pop("mouse") # Elimina y retorna inventario.clear() # Vacía el diccionario # Copiar diccionarios copia_inventario = inventario.copy()
