🎯 ¿Qué es la Programación?

💡 Concepto Fundamental

La programación es el arte de dar instrucciones precisas a una computadora para que resuelva problemas específicos. Es como escribir una receta muy detallada que la máquina puede seguir paso a paso.

🐍 ¿Por qué Python?

1

Fácil de Aprender

Sintaxis clara y legible, perfecto para principiantes

2

Versátil

Desarrollo web, ciencia de datos, inteligencia artificial

3

Demanda Laboral

Uno de los lenguajes más solicitados en la industria

🔍 ¿Qué está pasando aquí?

  • print(): Función que muestra texto en pantalla
  • Comillas: Indican que el contenido es texto (string)
  • #: Comentarios que explican el código
  • f-strings: Forma moderna de insertar variables en texto

🧠 Conceptos Fundamentales

🔤 ¿Qué es el Código?

El código es un conjunto de instrucciones escritas en un lenguaje que la computadora puede entender y ejecutar. Es como escribir una receta muy precisa.

📋 Elementos Básicos de un Programa:

1️⃣ Instrucciones

Comandos que le decimos a la computadora que ejecute

print("Hola")
# Instrucción: mostrar texto

2️⃣ Datos

Información que el programa procesa

"Hola" # Texto
25 # Número

3️⃣ Comentarios

Notas que explican qué hace el código

# Esto es un comentario
print("Hola") # Explica la línea

🎮 Ejercicio Interactivo: Identificar Elementos

En el siguiente código, identifica cada elemento:

📝 ejercicio.py
# Programa que calcula la edad en días
nombre = "Ana"

dias = edad_años * 365

¿Puedes identificar?

  • ✅ 1 comentario explicativo
  • ✅ 3 instrucciones de asignación
  • ✅ 1 instrucción de salida (print)
  • ✅ Datos: texto y números
